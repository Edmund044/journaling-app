# JOURNALING APP
 
To run the applciation
- 1.Clone the repo 
- 2.Make sure you have python and pip or pip3 installed
- 3.run the bash script command : `source start_backend.sh`
- NOTE : api documentation can be found on http://localhost:5000/apidocs/
